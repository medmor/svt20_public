Correction : 

R-1 – Les noms des étages bioclimatiques : 

  • e1 : Étage humide.
  • e2 : Étage Sub-humide.
  • e3 : Étage semi-aride.
  • e4 : Étage aride.
  • e5 : Étage saharien.

 

R-2 – Représentation des stations sur le diagramme. 


R-3 – On remarque sur le diagramme que la répartition du Thuya s’étale essentiellement sur les deux domaines : semi-aride et sub-humide, et il est moins présent sur les domaines : aride et humide. Il est totalement absent dans le domaine saharien.
On remarque aussi que le Thuya pousse dans les différents types d’hivers.
 

R-4 – Exemple de calcul de Q pour Azrou :

  • On sait queQ = 1000xPa / ((M + m) / 2) x (M – m))
  • Donc: Q = 1000 × 829 / ((305.7 + 275.4) / 2) x (305.7 – 275.4))
  • Q = 94.16 

R-5 – Représentation des stations sur le diagramme. 


R-6 – D’après le diagramme, le Thuya se trouve dans les régions de Azrou et Kénitra parce qu’elles sont incluses dans l’aire de répartition de Thuya, et il ne se trouve pas dans les autres régions (Ain Kahla, Errachidia, Kénitra).
