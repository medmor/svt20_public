Exercice :  Choisissez les propositions exactes :

  • 1 – L’écosystème est :
    • a – Un ensemble d’êtres vivants.
    • b – L’ensemble des constituants abiotiques du milieu.
    • c – La faune + la flore.
    • d – La biocénose + le biotope.  

 

  • 2 – Une courbe de fréquence unimodale :
    • a – Indique que le milieu étudié est homogène.
    • b – Indique que le milieu étudié est hétérogène.
    • c – Contient un seul mode.
    • d – Contient plusieurs modes

 

  • 3 – La faune est :
    • a – L’ensemble des êtres vivants dans un milieu.
    • b – Tous les animaux dans un milieu.
    • c – Tous les végétaux dans un milieu.
    • d – La biocénose.

 

  • 4 – On cite parmi les constituants abiotiques d’un milieu :
    • a – L’eau.
    • b – Les micro-organismes du sol.
    • c – Les arbres.
    • d – L’air.

 

  • 5 – On cite comme exemple de faune :
    • a – Le chêne-liège.
    • b – Les escargots.
    • c – Les lombrics.
    • d – Les oiseaux.

 

  • 6 – Les bonnes stations d’inventaire sont :
    • a – Les stations qui se trouvent dans la limite de deux milieux.
    • b – Les stations hétérogènes.
    • c – Les stations homogènes.
    • d – Les stations qui contiennent un nombre minimal d’espèces.

 

Correction :

  • 1 – d.
  • 2 – a, c.
  • 3 – b.
  • 4 – a, d.
  • 5 – b, c, d.
  • 6 – c.
