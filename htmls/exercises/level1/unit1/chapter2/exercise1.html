Exercice 1 : Choisissez les suggestions correctes.

  • 1 – Le sol qui a la plus grande capacité de rétention d’eau est :
    • a – Le sol sableux.
    • b – Le sol limono-sableux.
    • c – Le sol limoneux.
    • d – Le sol argileux.

 

  • 2 – Les grains les plus petits sont :
    • a – Les grains de sable.
    • b – Les grains de limon.
    • c – Les grains d’argile.
    • d – Les grains de gravier.

 

  • 3 – Les vers de terre :
    • a – Sont des organismes nuisibles pour le sol.
    • b – Ils contribuent dans l’aération du sol.
    • c – Sont des organismes bénéfiques pour le sol.
    • d – Ils provoquent le lessivage du sol.

 

  • 4 – Le sol calcaire :
    • a – est généralement basique.
    • b – est généralement acide.
    • c – Contient le CaCO₃.
    • d – Émet une effervescence en présence du vinaigre blanc.

 

  • 5 – Le sol :
    • a – ne contient pas d’êtres vivants.
    • b – contient seulement une flore.
    • c – contient seulement une faune.
    • d – contient une flore et une faune.

 

  • 6 – La structure compacte :
    • a – est caractéristique des sols riches en sable.
    • b – est caractéristique des sols riches en humus.
    • c – est caractéristique des sols riches en argile.
    • d – est la structure la plus favorable pour les végétaux.

 

  • 7 – Le sol le plus favorable pour les végétaux est :
    • a – le sol lessivé.
    • b – le sol qui a une structure particulaire.
    • c – le sol riche en humus.
    • d – le sol qui a une structure compacte.

 

  • 8 – L’eau utilisable par la plante :
    • a – est l’eau qui reste dans le sol après que le point de flétrissement est atteint.
    • b – est la capacité de rétention d’eau.
    • c – est la capacité de rétention d’eau + le point de flétrissement.
    • d – est la capacité de rétention d’eau – le point de flétrissement.

 

  • 9 – Les plantes calcicoles sont des plantes :
    • a – qui tolèrent le calcaire dans le sol.
    • b – qui ne tolèrent pas le calcaire dans le sol.
    • c – qui poussent dans des sols acides.
    • d – qui poussent dans des sols alcalins.

 

  • 10 – Un engrais NPK doit contenir :
    • a – l’azote.
    • b – le phosphore.
    • c – le potassium.
    • d – le magnésium.
